FlexLogger

A flexible, powerful, and beautiful logging library for Android applications

Get Started Documentation

Powerful Features

Flexible Configuration

Customize log levels, formats, and destinations with an intuitive Kotlin DSL. Configure once, log everywhere.

Multiple Destinations

Log to Logcat, files, or custom destinations. Built-in file rotation and crash-safe operations.

Pretty Printing

Beautiful JSON and XML formatting with automatic indentation and syntax highlighting in logs.

Thread Safe

Concurrent logging operations are handled safely across multiple threads without data corruption.

Smart Rotation

Automatic log file rotation based on size limits. Never worry about storage issues again.

High Performance

Optimized for minimal overhead with lazy evaluation and efficient string operations.

Documentation

Basic Setup 
// In your Application class
FlexLogger.init {
    enabled = true
    globalTagPrefix = "MyApp"
    minLevel = LogLevel.DEBUG
    showTimestamp = true
    showThreadInfo = true
    
    // Add destinations
    logcat()
    file(
        context = this@MyApplication,
        fileName = "app_log.txt",
        maxFileSizeMb = 5
    )
}

Usage Examples

Basic Logging 
class MainActivity : AppCompatActivity() {
    private val TAG = "MainActivity"
    
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        
        // Basic logging
        flexLogI(TAG) { "Activity created successfully" }
        flexLogD(TAG) { "Debug info: ${someVariable}" }
        
        // Error with exception
        try {
            riskyOperation()
        } catch (e: Exception) {
            flexLogE(TAG, e) { "Operation failed" }
        }
        
        // Pretty print JSON
        val jsonResponse = """{"user": "John", "id": 123}"""
        flexLogJson(TAG) { jsonResponse }
    }
}
Custom Destination 
class RemoteLogDestination : LogDestination {
    override val id = "remote_logger"
    
    override fun send(logMessage: LogMessage, formattedMessage: String) {
        if (logMessage.level >= LogLevel.ERROR) {
            sendToServer(formattedMessage)
        }
    }
}

// Add to configuration
FlexLogger.init {
    addDestination(RemoteLogDestination())
}
